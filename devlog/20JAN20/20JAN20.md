## markdown-converter
My latest and greatest project. Written in Lua (my favorite programming language), it takes input from a `.md` or a `.markdown` file and converts it into valid HTML. Fun fact, this very page was written in markdown. You're now seeing it in all it's glory thanks to the tool. There are a few features I'd like to finish however, being **list support** and **a command-line interface**. See it in action [here](https://github.com/taizweb/markdown-converter)!
