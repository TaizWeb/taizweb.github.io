It's Been a While...

April 30th, 2020

University has eaten up nearly all my spare-time this past month, but as of tomorrow I'll be done with class aside from some final exams. So how am I going to spend my summer? Aside from work, I'll try my hand at some gamejams, in particular itch's Metroidvania Month Jam. I'm pretty excited for this, and I've already begun planning a game. I'm not going to finish Project Neutron and send that in, rather, I'll make an entirely new game: Project Proton.

Bitter Jam

Points for originality, right? Though that's not the only jam I'm planning on doing. In fact, starting Friday, I plan to partake in Bitter Jam. The idea behind Bitter Jam is that you're given a spritesheet when it starts, and it's your goal to make something cool out of it. I figure this will be a good "first game" to actually release since I can have a lot less work to do, and I can focus on coding the entire project instead of trying to make assets.

Heartbeat

Future games aside, I've worked on a few things since the last devlog. For starters, I began creating Heartbeat, a minimalist library to help with Project Proton. It currently can draw entities and tiles to the screen, along with basic collisions for them. It's not very far along yet, so I doubt I'll be using it for Bitter Jam (nor am I going to make yet another metroidvania).

fedi-lua

I also began writing fedi-lua a library to interact with the Fediverse. It's not much either, but it can currently create posts and attach images/polls to them. I may spin up a bot with it, but we'll see.

Project Neutron

I also added a very basic menu to Project Neutron, and rewrote my code to have things be more re-usable. I also added item pickups, and gave the player an inventory viewable from the menu. I also stomped some minor bugs in the codebase.

Blog

And I also wrote a post on The weirdness of Lua's string handling, and released a little fix for it. This was something I needed to create in order for fedi-lua to work properly. Earlier in the month, I added a dark theme option to the site. This can be found in the sidebar on desktop, and the bottom of the page on mobile.

That's about all I did this month, hopefully I'll be able to be more active now that university isn't eating up all my spare-time anymore. I'd also like to go back to the weekly schedule of updates. Speaking of updates, I also plan to release a long-ish devlog on the creation of my game for Bitter Jam, so definitely look out for that on Monday. Thanks for reading!
