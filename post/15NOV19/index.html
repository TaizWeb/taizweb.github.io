Taiz 2.0

November 15th, 2019

As you’ve all probably noticed by my dead social media and Github, I’ve been away for a while. This past years been a big change for me, and it seems like things are looking up. I’m finally going to college (albeit a year later) and for the most part I’m in a good place financially in part thanks to my new job I’m working.

College has been quite the experience for me. I really don’t have as much free time as I used to, but I’m fine with that. What time I do have goes to spending time with my significant other and friends, along with some programming projects on the side. I’ve had numerous ideas of things to build, and to commemorate my “revival” I’ve moved some of the things I worked on last year over to my github. One such project is lua-tetris, my first experiment with love2d, that I made over my last winter break. Be warned, the love2d engine has since updated, so it’s very buggy until I get around to refactoring it. There’s another love2d project I’ve been working on as well, that being testgame_v2, called which because I wasn’t entirely certain what I was making, aside from a simple 2d platformer.

At the moment, I’m not sure if I’ll manage to find the time to fix and update them, although I really consider testgame to be the start of my first big project. Testgame_v2 is planned to be a metroidvania-type game, I’ve already written much of the story and some of the gameplay elements. My main drawback is that I can’t draw, so I wouldn’t expect it to be finished anytime soon. Lua-tetris is an easier matter, since the game itself is pretty much complete aside from a couple new bugs thanks to the new API changes.

One of the projects more in the spotlight, however, is this little blog. I’ve built some blogging software before, sure, but it relied on having a server with nodejs installed, a SQL server, and a whole bunch of dependencies. I spent more time trying to get the backend up and going than actually writing on it. So I’ve decided that this blog will be a simpler approach. At the moment, I’m writing this in LibreOffice Writer, with intentions to convert it over to html when I’m done. I doubt I’ll end up using other services to run this, hell, I’ll probably just write my own converter sometime.

As for the frontend, I have a few rules in place. For starters, I don’t want to use any sort of framework. The web nowadays is so bloated with trackers, popups, and other needless things that I wanted to take a step back. I’ll use some css, and work my html from templates. Javascript will be used incredibly sparingly, if at all. It’ll be a bit of a challenge, seeing as how if I want to update the UI, I’ll need to manually update every single post to match. Not much of an ideal solution, but I suppose it’ll work until I think up something better.

As for post frequency, I really can’t promise much. I’m hoping to push out a new post on a weekly basis, but depending on how busy I get over at college that might not be possible. Another thing is keeping users updated on when I do post new things. Once again, not having that server-side aspect really complicates things. I’ve considered simply making a twitter (or fediverse) account to notify users when I publish, but for right now the only way is to bookmark this site and check back occasionally.

Speaking of college life, I’m currently studying Computer Science, something I’ve been interested in for a long time now. The coursework isn’t that bad, as I already know the majority of the computer concepts thrown at me, so I’m using that spare-time to learn Java, as I’m certain I’ll be expected to use it extensively later in my degree. College itself is rather pricey since I’m currently living on dorms, but next year I’m planning on moving in with my significant other to cut costs. My entire tuition is covered, luckily, and I also receive some pocket money thanks to my new position.

That’s it for my first post, at the moment I already have numerous ideas for my next ones. I can say with a fair amount of certainty that this won’t just fizzle out like some of my other endeavors (though to be fair I did disappear for like a year). Thanks for reading.
